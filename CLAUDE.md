# CLAUDE.md — контекст проекта для Claude Code

Положи этот файл в корень проекта. Claude Code будет читать его автоматически в каждой сессии.

---

## О проекте

MVP веб-приложения: ИИ-консультант по поступлению в университеты (США и Великобритания). Цель — проверка спроса, поэтому скоуп сознательно минимальный.

## Владелец проекта

Не программист. Объясняй решения простым языком, избегай жаргона без пояснений. Перед сложными изменениями — сначала план, потом код.

## Стек

- Next.js (App Router) + TypeScript + Tailwind
- Prisma + SQLite
- Anthropic API (claude-sonnet-4-6), стриминг через Vercel AI SDK
- Деплой: Vercel

## Ключевые файлы

- `prompts/advisor.md` — system prompt ИИ-консультанта. НЕ редактируй его без явной просьбы.
- `app/api/chat/route.ts` — единственная точка вызова Claude API
- `.env` — секреты. Никогда не выводи содержимое и не коммить в git.

## Железные правила

1. API-ключ только на сервере. Любой код, тянущий ANTHROPIC_API_KEY на клиент — ошибка.
2. Не добавляй функционал сверх MVP по своей инициативе (авторизация, оплата, база вузов — всё это потом).
3. История диалога в API-запросах ограничена 20 последними сообщениями — не убирай это ограничение, это контроль расходов.
4. Комментарии в коде на русском.
5. После каждого изменения — короткая инструкция, как проверить, что оно работает.

## Команды

- `npm run dev` — локальный запуск
- `npx prisma studio` — посмотреть базу данных
- `npx prisma migrate dev` — применить изменения схемы БД
